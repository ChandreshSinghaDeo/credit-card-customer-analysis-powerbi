# Credit Card Financial Analysis Dashboard (SQL + Power BI)

## 📊 Project Overview

This project analyzes credit card customer data to understand spending behavior, revenue trends, and customer segmentation using SQL and Power BI.

## 🎯 Objectives

* Analyze customer spending patterns
* Identify high-value customers
* Track revenue trends
* Support business decision-making

## 🛠 Tools Used

* SQL (Joins, Aggregations, Data Extraction)
* Power BI (Dashboard & Visualization)
* Power Query (Data Cleaning)

## 🗄️ SQL Work

* Extracted data using SQL queries
* Prepared dataset for Power BI

## 📈 Key Insights

* Top customers contribute a major share of revenue
* Regional trends show variation in spending behavior
* Monthly trends indicate seasonal patterns
